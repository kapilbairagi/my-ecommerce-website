### ❌ Problem

* Products uploaded via the **admin panel** were **not saved persistently**.
* Uploaded data existed only in memory and **disappeared after a page refresh**.
* **Home page sections** like `featured`, `popular`, and `new-arrivals` were **not updating** with new products.
* **Shop page** did not reflect newly added items.
* **Offers page** remained static and did not display relevant uploaded products.
* Product **targets and categories** were not being used correctly to filter content on each page.
* There was **no unified data source** connecting the admin uploads with the customer-facing pages.
